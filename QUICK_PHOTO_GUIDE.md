# Quick Photo Management Guide

## 30-Second Overview

Manage vehicle photos directly in admin portal before posting to website.

---

## Quick Start

### 1. Edit Vehicle
- Go to Admin → Vehicles
- Click "Edit" on vehicle

### 2. Manage Photos
- Click "Manage Photos" button
- Section expands

### 3. Add Photo
- Paste photo URL
- Click "Add" or press Enter
- Photo appears in list

### 4. Reorder
- Use ↑ ↓ arrows to move photos
- First photo = main display photo

### 5. Set Main
- Click "Main" button on any photo
- Moves to first position

### 6. Remove
- Click "X" to delete photo

### 7. Save
- Click "Save" button
- Changes sync to website

---

## Photo Controls

| Button | Action |
|--------|--------|
| **Add** | Add new photo URL |
| **↑** | Move photo up |
| **↓** | Move photo down |
| **Main** | Set as main photo |
| **X** | Delete photo |
| **Save** | Save all changes |

## Website Display

- Main photo on vehicle card
- Full gallery in modal
- Zoom on hover
- Thumbnail navigation
- Photo counter

---

## Troubleshooting

**Photo not showing?**
- Check URL is correct
- Verify image exists
- Refresh website

**Can't reorder?**
- Make sure in edit mode
- Use arrow buttons
- Save changes

**Changes not syncing?**
- Click "Save" button
- Refresh website
- Clear cache

Add, remove, reorder, and manage photos before posting vehicles. All changes sync to website instantly! 📸
