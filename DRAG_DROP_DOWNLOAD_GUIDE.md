# Drag-and-Drop & Download Features Guide

## New Features

Your photo management system now includes two powerful features:

### 1. 🎯 Drag-and-Drop Photo Reordering
Easily reorder photos by dragging them to new positions.

### 2. ⬇️ Download Photos to Computer
Download any photo directly to your computer with a single click.

---

## Drag-and-Drop Feature

### How to Use

1. **Edit a Vehicle**
   - Go to Admin → Vehicles
   - Click "Edit" on vehicle

2. **Open Photo Manager**
   - Click "Manage Photos" button

3. **Drag Photos**
   - Click and hold the grip handle (≡) on any photo
   - Drag to new position
   - Release to drop
   - Photo reorders instantly

### Visual Feedback

- **Grip Handle** - Shows draggable area (≡ icon)
- **Dragging** - Photo becomes semi-transparent (50% opacity)
- **Drop Target** - Hover area highlights
- **Main Photo** - Red border indicates first photo

### Tips

✅ Drag from the grip handle for best results
✅ Photos reorder in real-time
✅ First photo becomes main display photo
✅ Combine with arrow buttons for precise control
✅ No need to save until done with all changes

### Example Workflow

```
1. Drag photo 3 to position 1
   ↓
2. Drag photo 5 to position 2
   ↓
3. Drag photo 2 to position 3
   ↓
4. Click "Save" to save all changes
```

## Download Feature

### How to Use

1. **Edit a Vehicle**
   - Go to Admin → Vehicles
   - Click "Edit" on vehicle

2. **Open Photo Manager**
   - Click "Manage Photos" button

3. **Download Photo**
   - Click download icon (⬇️) on any photo
   - Photo downloads to your computer
   - File named: `VehicleName-photo-#.jpg`

### Download Details

**File Naming**
- Format: `{Brand}-{Model}-photo-{Number}.jpg`
- Example: `Honda-Civic-photo-1.jpg`
- Example: `Toyota-LC200-photo-3.jpg`

**File Location**
- Downloads to your default Downloads folder
- Can be moved anywhere after download

**Supported Formats**
- JPG / JPEG
- PNG
- WebP
- GIF

### Use Cases

✅ Backup photos to external drive
✅ Share photos with team members
✅ Edit photos in external software
✅ Archive vehicle photos
✅ Transfer to another computer

### Troubleshooting

**Download Failed?**
- Check internet connection
- Verify photo URL is accessible
- Try copying URL instead
- Check browser download settings

**File Not Appearing?**
- Check Downloads folder
- Check browser download history
- Verify file permissions
- Try different photo

## Tips & Tricks

### Pro Tips

💡 **Drag Multiple Times**
- Drag same photo multiple times to fine-tune position
- No limit on reordering

💡 **Download Before Deleting**
- Download photos before removing them
- Backup important photos

💡 **Organize by Angle**
- Front, side, rear, interior, details
- Drag to arrange logically

💡 **Use Arrow Buttons Too**
- Arrows work for precise single-step moves
- Drag for larger jumps

💡 **Save After All Changes**
- Make all changes first
- Save once at the end
- More efficient

### Keyboard Shortcuts

- **Enter** - Add photo URL (when in input field)
- **Drag** - Reorder photos (mouse)
- **Click** - All other actions

## Troubleshooting

### Drag-and-Drop Not Working

**Problem**: Can't drag photos
**Solution**:
1. Make sure you're in edit mode
2. Click and hold the grip handle (≡)
3. Drag to new position
4. Release to drop
5. Try different browser if still not working

### Download Not Working

**Problem**: Download button doesn't work
**Solution**:
1. Check internet connection
2. Verify photo URL is accessible
3. Check browser download settings
4. Try different photo
5. Check browser console (F12) for errors

### Photos Not Reordering

**Problem**: Drag doesn't reorder photos
**Solution**:
1. Make sure in edit mode
2. Use grip handle (not photo itself)
3. Drag to different position
4. Release to drop
5. Refresh page if stuck

### Download File Corrupted

**Problem**: Downloaded file won't open
**Solution**:
1. Try downloading again
2. Try different photo
3. Check file extension
4. Try opening with different app
5. Copy URL and download manually
