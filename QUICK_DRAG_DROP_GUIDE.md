# Quick Drag-and-Drop & Download Guide

## 30-Second Overview

Drag photos to reorder and download them to your computer.

---

## Drag-and-Drop

### How to Drag

1. Edit vehicle
2. Click "Manage Photos"
3. Click and hold grip handle (≡)
4. Drag to new position
5. Release to drop
6. Photo reorders instantly

### Visual Cues

- **Grip Handle** - ≡ icon (draggable area)
- **Dragging** - Photo becomes faded
- **Main Photo** - Red border on first photo

---

## Download

### How to Download

1. Edit vehicle
2. Click "Manage Photos"
3. Click download icon (⬇️) on photo
4. File downloads to computer
5. Named: `VehicleName-photo-#.jpg`

### File Location

- Downloads to your Downloads folder
- Can be moved anywhere
- Backup to external drive

---

## Controls

| Icon | Action |
|------|--------|
| **≡** | Drag to reorder |
| **↑** | Move up |
| **↓** | Move down |
| **Main** | Set as main |
| **⬇️** | Download |
| **X** | Delete |

---

## Tips

✅ Drag from grip handle
✅ First photo = main display
✅ Download before deleting
✅ Save after all changes
✅ Works on mobile too

---

## Troubleshooting

**Can't drag?**
- Use grip handle (≡)
- Make sure in edit mode

**Download failed?**
- Check internet
- Try different photo
- Copy URL instead

**Photos not reordering?**
- Refresh page
- Try different browser

---

Drag to reorder, download to backup! 📸
